Para colaborar con el proyecto primero debes:
1. Crear el fork del repositorio principal
2. Registar una tarea como issue
3. Asignarse la parea a uno mismo
4. Clonar el repositorio remoto propio a tu maquina local
5. Implementar los cambios necesarios para la tarea (issue)
6. Confirmar los cambios (commit)
7. Enviar los cambios realizados en forma local al repositorio remoto propio
8. Enviar un pull-request con los cambios
9. Agregar nuevas issues y repetir el proceso
